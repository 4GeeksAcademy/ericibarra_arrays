# `05.2` Loop from the top

Este loop o bucle está haciendo un bucle al arreglo o array de principio a fin ... aumentando uno por uno.

## 📝 Instrucciones:

1. Intenta hacer un bucle desde el final hasta el principio.

## Resultado esperado:

